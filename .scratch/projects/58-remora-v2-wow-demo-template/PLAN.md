## Goal
Create a new numbered project template directory, capture high-impact remora-v2 demo concepts, and deliver two execution-ready runbooks (primary + backup).

## Steps
1. Create project scaffolding under `.scratch/projects/58-remora-v2-wow-demo-template/`.
2. Populate required standard files (`PLAN`, `ASSUMPTIONS`, `PROGRESS`, `CONTEXT`, `DECISIONS`, `ISSUES`).

5. Produce a concrete primary runbook for "Event Storm Control Room".
6. Produce a backup runbook that remains usable when live model interaction is unstable.

## Acceptance Criteria
- New numbered project directory exists.

- A demo brainstorming document exists with actionable, technically grounded concepts.
- A ranked shortlist and recommended first demo are documented.
- A primary runbook exists with exact setup and live script commands.
- A backup runbook exists with a deterministic fallback demo path.
