# Assumptions — 63-web-ui-improvement-template

1. The UI work targets the existing Remora web app in `src/remora/web/static/index.html` and related API routes.
2. Primary audience is technical users using localhost workflows.
3. Improvements should preserve existing runtime behavior and API compatibility unless explicitly changed.
4. Accessibility, readability, and observability of graph/activity data are first-order goals.
5. Visual polish should not come at the cost of responsiveness on medium-size graphs.
